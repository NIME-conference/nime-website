sutoolz 1.0 alpha : 3D Software Music Interface

Jordan Wynnychuk, Richard Porcher, Lucas Brajovic, Marko Brajovic, and Nacho Platas

Proceedings of the International Conference on New Interfaces for Musical Expression

  • Year: 2002
  • Location: Dublin, Ireland
  • Pages: 154–155
  • Keywords: 3D music interface, 3D sound, analogue input controllers, audio localization, audio visualization, digital architecture, hybrid environments, video game navigation
  • DOI: 10.5281/zenodo.1176478 (Link to paper)

Abstract:

The demo sutoolz 1.0 alpha is a 3D software interface for music performance. By navigating through a 3D virtual architecture the musician uses a set of 3D tools to interact with the virtual environment: gameplay zones, speaker volumes, speaker volume membranes, speaker navigation volumes and 3D multi-band FFT visualization systems.
